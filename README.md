# **一个简易但功能还行的工资信息管理系统**

## **用户系统**

分为管理员和用户,管理员初始密码会在程序第一次启动时简单加密后写入文件.用户则无需输入密码,直接使用.

## 功能介绍

### 管理员:

1:录入员工数据.

2:浏览和查询数据.

3:修改写入的数据.

4:对数据进行一定的处理(统计和排序).

5:修改密码.

### 员工:

1:查询数据.

2:对数据进行统计后查看.

## 使用方法

直接下载后用vs打开编译运行即可,没试过别的编译器,把前面的宏定义删了后,应该是通用的.(管理员初始密码为777)
